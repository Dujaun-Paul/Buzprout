# Buzprout — IP & Handover Checklist

Complete when client pays **final milestone**, **final Flex installment**, or **Build + Care buyout**.

**Client:** ___________________ **Project:** ___________________ **Date:** __________

## Final payment

- [ ] Final invoice paid in full
- [ ] No outstanding Flex or retainer arrears (or documented settlement)

## Legal

- [ ] Signed IP assignment / ownership transfer document
- [ ] Flex Schedule marked complete (if applicable)
- [ ] Client confirms receipt of handover pack

## Technical handover

- [ ] Source code repo transferred to client org (or export delivered)
- [ ] Hosting account migrated OR credentials documented
- [ ] Domain transferred or client registrar access provided
- [ ] Environment variables / secrets rotated and handed to client securely
- [ ] Third-party accounts (Stripe, email, analytics) ownership confirmed

## Documentation

- [ ] Admin login credentials delivered
- [ ] Short Loom or written guide for common tasks (optional but recommended)
- [ ] Known limitations / future recommendations noted

## Post-handover care offer

- [ ] Care Essential / Standard quoted (hosting from JMD 5,000/mo)
- [ ] OR Systems retainer quoted for software clients
- [ ] Client decision recorded: ☐ Care ☐ Retainer ☐ Self-manage
